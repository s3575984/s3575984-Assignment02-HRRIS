Algorithm for inserting a record in the booking table

When click on insert button in form booking

Start function validateForm

If validateForm return true then

    Initiate object oBook in class addToDb

    Call sub insertBooking, which require a hashtable of data from sub getFormData

    Initiated sub getFormData, result is put all data in the form into a hashtable (with key are : booking_id, datebook, room_id, customer_id, num_days, num_guests, checkin_date, total_price, comments, each key has a value)

    Initiate sub insertBooking, using from the hashtable for parameters adding to the query, then a connection to MS access database is create and open, and the query will be send to the database, and successful message will be shown if query is successfully executed, and error in message box is shown if there are error/exception occurs during the working process of the sub

Else

    A message box show there is/are invalid input(s)

If there are any exception occurs, a message box will show the error.

How to insert new booking

    You can either select "Book A Room" button from the menu form at the beginning or select Menu > Data Capture > Booking

    (If you select "Book A Rooom" button, please skip this step) Select "New" button on the top of the group of buttons on your right

    Fill in all the fields of the form (Excepting the Customer ID field, because it will not be recorded)

    Choose "Insert" button after you have filled all the required fields. The system will show a message if there are any invalid input

    A mesaage box "Insert successfully" will be displayed if the record is inserted
